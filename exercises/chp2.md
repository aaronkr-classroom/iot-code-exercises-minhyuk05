# chp2 연습문제

2. 마이크로파이썬의 펌웨어를 올리기 위해서 부팅할 때 별도의 버튼을 클릭하게 합니다. 해당 절차의 필요성을 설명하세요.

   마이크로파이썬 펌웨어를 업로드할 때 부팅 시 버튼을 누르는 이유는
    디바이스를 일반 실행 모드가 아닌 펌웨어 업로드 모드(부트로더 모드)로 진입시키기 위함이다.
    
    필요성:
    1.기존 프로그램 실행 방지
     - 기존 코드가 실행되면 펌웨어 업로드 방해
    2.펌웨어 쓰기 권한 확보
     - 플래시 메모리에 접근 가능한 모드로 전환
    3.안정적인 업로드 환경 제공
     - 충돌 없이 펌웨어를 정확하게 기록
    4.USB 인식 모드 활성화
     - PC에서 저장장치처럼 인식 가능
